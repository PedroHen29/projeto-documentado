#  Sistema de Farmácia

## Descrição
O **Sistema de Farmácia** é uma aplicação desenvolvida para gerenciar o funcionamento de uma farmácia de forma eficiente. O projeto permite o controle de estoque de medicamentos, cadastro de clientes e funcionários, registro de vendas e acompanhamento de relatórios.

Seu principal objetivo é facilitar a organização e automação dos processos internos, reduzindo erros e melhorando o atendimento ao cliente.

## Instalação

### Requisitos
- Node.js instalado
- Banco de dados (MySQL ou PostgreSQL)
- Git

### Passos para instalação ###
```bash
git clone https://github.com/usuario/farmacia.git
cd farmacia
npm install.
```
# Exemplo de funcionalidades
``` java script
function cadastrarMedicamento(nome) {
  console.log("Medicamento cadastrado: " + nome);
}

cadastrarMedicamento("Paracetamol");
```

# Funcionalidades Principais

- Cadastro de medicamentos
- Controle de estoque
- Registro de vendas
- Cadastro de clientes
- Relatórios de vendas
